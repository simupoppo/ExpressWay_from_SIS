# Express Way from Project SIS
本アドオンは、SISプロジェクト https://github.com/128na/sis 規格を利用した高速道路用アドオンです。  
基本的にSISプロジェクトの高速道路規格、及び使用法に準拠していますが、一部アドオンではさらに複雑な設置方法となっています。
